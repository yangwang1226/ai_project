# AI销售练习平台 - C端前端原型

一个现代化、科技感十足的AI销售训练平台C端前端静态原型。

## ✨ 功能特性

### 📱 页面展示

1. **首页（Home）**
   - 现代化英雄区域
   - 轮播广告展示
   - 3D浮动动画效果
   - 特性介绍
   - CTA行动号召

2. **注册页面（Register）**
   - 时尚科技风设计
   - 手机号验证码注册
   - 左右分栏布局
   - 浮动几何图形装饰

3. **场景选择（SelectScene）**
   - 三大行业场景：教育、房地产、汽车
   - 卡片式选择交互
   - 难度指标显示
   - 进度条导航

4. **声音配置（SelectVoice）**
   - 客户性别选择（男/女）
   - 性格选择（冷漠/适中/积极）
   - 实时预览效果
   - 性格强度指示器

5. **练习页面（Practice）**
   - 左侧：客户形象动画区
     - 音频波形动画
     - 说话状态指示
     - 通话控制按钮
   - 右侧：对话记录区
     - 实时对话展示
     - AI建议提示
     - 快捷回复按钮
   - 顶部：计时器、状态、得分
   - 练习总结弹窗

## 🎨 设计亮点

- **科技感配色**：深色主题 + 蓝紫渐变
- **流畅动画**：淡入、滑入、浮动等多种动画
- **响应式设计**：完美适配桌面和移动端
- **现代化UI**：卡片式布局、毛玻璃效果、渐变按钮
- **交互反馈**：悬停效果、点击反馈、状态指示

## 🚀 快速开始

### 安装依赖

```bash
cd ai_practice_client
npm install
```

### 运行开发服务器

```bash
npm run dev
```

访问：http://localhost:5173

### 构建生产版本

```bash
npm run build
```

## 💡 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vue Router** - 官方路由管理
- **Pinia** - 状态管理（预留）
- **Vite** - 新一代前端构建工具
- **纯CSS** - 无UI框架依赖，完全自定义

## 🎨 设计规范

### 颜色系统

- **主色**：`#6366f1` (科技蓝)
- **渐变**：`linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- **背景**：`#0f172a` (深色)
- **卡片**：`#1e293b`
